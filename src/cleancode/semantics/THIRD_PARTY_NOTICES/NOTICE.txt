package because `embeddings.npz` (in the parent directory) is a derivative

weights, inference code, or training code are included or redistributed here
— only a derived, fixed-size table of word vectors, frozen and used solely as
a static feature lookup for the CM307 docstring/comment classifier. It is not
used, and must not be used, to train or improve any other large language
model.
